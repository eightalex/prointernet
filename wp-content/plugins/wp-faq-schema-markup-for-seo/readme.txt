2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How to verify if my FAQ Schema is working? =

Go to [Google Snippet Tool](https://search.google.com/structured-data/testing-tool/),
add your URL and verify the FAQ Schema

= What is FAQ Meta column in posts table? =

FAQ meta column shows status of the FAQ Schema added to your post and pages.
* **Grey** for No schema added
* **Red** Schema added but no FAQs added.
* **Orange** Less than three FAQs added.
* **Green** Greater than pr equal to three FAQs added.
